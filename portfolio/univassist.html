프로젝트 소개

서비스 목적·플랫폼·무드·키워드를 입력하면 WCAG 2.1 기준을 충족하는 컬러시스템을 AI가 자동으로 생성합니다. 색각 이상자(적록색약) 시뮬레이션과 명도 대비를 실시간으로 검증하며, 웹폰트 페어링 추천과 플랫폼별 타이포그래피 가이드까지 하나의 도구로 제공합니다.

숙명여자대학교 교육대학원 AI융합교육학과 석사 논문 연계 프로젝트로, 디자인을 배우는 학습자가 접근성 원칙을 실습 환경에서 직접 체험할 수 있도록 설계했습니다.

개발 배경

마이스터고 디자인 수업에서 학생들이 색채 시스템을 설계할 때 접근성 기준을 체감하기 어렵다는 문제에서 출발했습니다. 이론으로만 배우던 WCAG 기준을 실시간으로 확인하고, 색각 이상자 시각으로 직접 시뮬레이션해 볼 수 있는 도구가 필요했습니다. AI를 활용해 초안을 빠르게 생성하고, 학생이 직접 조정하는 방식으로 유니버설 디자인 감각을 키울 수 있도록 설계했습니다.
